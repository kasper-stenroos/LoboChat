LoboChat

RESTful Java Chat Application

Download Download View LoboChat on GitHub

Overview

School project for Application Development Methods course. The target environment is a fictional hospital and the user group consists of doctors, nurses and other personnel.

Technical Details

The back-end is implemented with Java using Jersey framework and the front-end is based on a web client implemented with HTML, CSS, JavaScript, jQuery and Ajax. The notification system is based on web sockets. The application has optimized UIs for mobile, tablet and desktop use.
